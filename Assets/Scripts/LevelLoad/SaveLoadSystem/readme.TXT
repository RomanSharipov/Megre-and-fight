Это пак для сохранения/загрузки структур.
Для чего можно использовать: в целом что угодно, что хотите сохранить, например прокачка персонажа.
Как работает - сохраняет в бинарный файл с расширением .dat. Работает быстро, можно по несколько раз в секунду сохранять, 
ничего не потеряется.
Файл лежит по адресу:
C:\Users\<Имя пользователя>\AppData\LocalLow\<Название компании>\<Название проекта>\<Имя файла>.dat

Как пользоваться:
1. Основной файл - класс SaveLoadSystem<T>.
2. Там, где нужно сохранить/загрузить штуки - создаете его экземпляр, в конструктор нужно пробросить имя файла строкой;
3. Для сохранения нужно передать структуру типа T в метод SaveData;
4. Для загрузки - вызвать метод GetLoadedData, вернет структуру типа T с данными из файла, если файла нет - создаст через пустой конструктор;
5. Так как у нас C# 8.0, то в нем не доступно переопределение пустого конструктора. Поэтому всегда проверяйте, что он вернет.
6. Если нужно закинуть дефолтные данные - делайте после получения файла.

В папке Example есть тестовая сцена с реализацией сохранения/загрузки и тестовая структура.

Q/A:
1. Можно ли сохранить класс - нет, нельзя, работает только со структурой. Если нужно сохранять классы - убирайте из 
SaveLoadSystem "where T : struct", но что будет работать не обещаю.
2. Можно сохранить свои типы данных? - Можно, пометьте атрибутом [Serializable]
